# Roadmap

Indic Dictation aims to become a low-latency macOS dictation layer for Indic-language speakers who want final English text inserted into any app.

## Now

- Marathi-to-English streaming dictation.
- Native macOS menu bar app.
- Global shortcut recording.
- Hold-to-record and double-tap lock modes.
- Mic selection from the menu bar.
- Optional live preview.

## Next

- Configurable source language selection.
- Better first-run onboarding for macOS permissions.
- Signed release artifacts and simpler installation.
- Latency diagnostics visible in a user-friendly way.
- More robust network and API-key error messages.

## Later

- Support and documentation for more Sarvam-supported Indic languages.
- Optional text cleanup presets, such as concise, professional, or email-ready.
- Automated regression tests for shortcut behavior and streaming response handling.
- Better accessibility for the floating voice indicator.
- Community examples and troubleshooting guides.
